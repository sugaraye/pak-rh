# PAK Carrière RH 2026

Déploiement recommandé :
1. Créer un projet Supabase.
2. Exécuter sql/schema.sql.
3. Créer les buckets : cv, diplomes, formations, identites, accuses.
4. Dans Netlify > Site configuration > Environment variables :
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
   - SUPABASE_SERVICE_KEY
5. Déployer le dépôt GitHub sur Netlify.

Cette version est un socle professionnel à compléter avec vos données RH.
